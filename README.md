🌐 وصف المشروع (Portfolio Website)

هذا المشروع عبارة عن موقع بورتفوليو شخصي (Personal Portfolio Website) تم تطويره باستخدام تقنيات الويب الحديثة، بهدف عرض المهارات والخبرات والمشاريع بشكل احترافي ومنظم.

الموقع مصمم ليكون واجهة تعريفية بالمطور، ويحتوي على أقسام أساسية مثل:

نبذة عني (About Me)
المهارات التقنية (Skills)
المشاريع (Projects)
وسائل التواصل (Contact)
⚙️ التقنيات المستخدمة:
HTML5
CSS3
JavaScript
GitHub Pages للاستضافة
🎯 هدف المشروع:

عرض الهوية المهنية للمطور بطريقة بسيطة وسريعة، وتسهيل الوصول إلى المشاريع وروابط التواصل، مع تصميم متجاوب يعمل على جميع الأجهزة.

مميزات
